## Embedding model
В качестве модели эмбеддингов, самый жир брать обученную на русском с ориентированием в медицину, но я не нашел такую(если найдете, добавляйте).

Я нашел [BERT-подобную модель](https://huggingface.co/sergeyzh/LaBSE-ru-turbo). Это легковесная модель общего назначения, для русского языка.

## Re-rank model
Для этой модели часто рекомендуют cross-encoder. Чтобы модель была для медицины и русского, я тоже не нашел. Взял [эту](https://huggingface.co/ncbi/MedCPT-Cross-Encoder), с надеждой, что медицинские термины могут быть важнее.
